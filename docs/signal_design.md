# News & Fed Sentiment — 신호 설계 문서 (초안 v0.1)

> **상태: 제안(초안).** 팀·지도교수 검토·합의 후 확정. 임계값(θ)은 모두 **잠정값**이며 데이터로 보정 예정.
> 근거: Phase 5 프로토타입 분석 (FOMC 톤 ↔ S&P500/VIX, 2000–2021, 187회의).

---

## 0. 목적과 범위

**목적**: FOMC·뉴스의 감성(톤)과 시장 지표를 결합해, 연구자·투자자에게 **참고용 위험/기회 "알림"과 그 신뢰도**를 제공한다.

**무엇인가 / 무엇이 아닌가**

| 이다 | 아니다 |
|---|---|
| 연구·의사결정 **참고 정보** | 투자 조언·매매 지시 |
| 톤과 시장의 **정합/괴리 알림** | "사라/팔아라" 신호 |
| **신뢰도·한계를 동봉**한 정직한 정보 | 수익 보장·예측 단언 |

---

## 1. 설계 원칙 (프로토타입 발견 근거)

Phase 5 분석에서 확인된 사실이 설계를 강제한다:

1. **톤은 당일 주가 "방향"을 예측하지 못한다** (상관 −0.09 ≈ 0).
   → ❌ "톤→수익률 예측" 신호는 만들지 않는다. (효율시장·금리서프라이즈 지배)
2. **톤은 시장 "공포 수준(VIX)"과 동행한다** (상관 −0.34).
   → ✅ 톤을 **환경 동행지표**로 사용.
3. **톤↔시장 괴리는 위기에 집중된다** (2020 COVID, 2008 GFC).
   → ✅ **괴리 탐지를 핵심 신호**로.
4. **표본이 작다** (FOMC ~170회).
   → 복잡한 학습 금지. **단순·해석가능한 규칙 + 벤치마크 비교 필수**.

## 3. 신호 정의 (3종)

각 신호는 **조건 → 출력 → 기간**을 명확히 한다. (θ = 보정 대상 잠정값)

### 신호 A — 톤 급변 (Tone Shift)
- **근거**: 학계상 FOMC 텍스트의 정보는 "절대 수준"보다 **직전 대비 변화**에 있다.
- **조건**: |이번 회의 톤 − 직전 회의 톤| ≥ θ_shift (잠정 0.2)
- **출력**: 🔼 톤 개선 / 🔽 톤 악화 (+ 변화량)
- **기간**: 발표일 기준

### 신호 B — 톤↔시장 괴리 (Divergence) ⭐ 핵심
- **근거**: 괴리가 위기·전환점에 집중 (프로토타입 확인).
- **조건**: sign(톤) ≠ sign(시장 반응) **그리고** |톤| ≥ θ_t **그리고** |시장 반응| ≥ θ_m
- **출력**: ⚠️ 괴리 경고 (예: "연준 톤 긍정인데 시장 급락")
- **기간**: 발표 후 N일 (잠정 N=2, ET 발표시각 정렬)

### 신호 C — 톤↔공포 정합 이탈 (Tone vs VIX)
- **근거**: 평소 톤−VIX는 음의 동행. 이 관계가 깨지면 이례.
- **조건**: 톤 긍정인데 VIX 급등 (또는 반대) — 동행 관계 이탈
- **출력**: 🟢 정합 확인 / ⚠️ 동행 이탈
- **기간**: 발표일 ± 윈도우

**종합 출력 등급**: 🟢 정합 · ⚪ 중립 · ⚠️ 주의 · 🔴 경고 (+ 신뢰도)

## 4. 검증 방법 (필수)

신호를 발표하기 전, **반드시** 과거 데이터로 검증한다.

**벤치마크 비교** — "이 신호가 추가 정보를 주는가?"
1. 단순보유 (buy-and-hold)
2. **금리 결정만 본 경우** ← 핵심: 톤이 금리결정 너머 정보를 주나?
3. 무작위 신호

**지표**
- 신호 적중률 (signal hit rate)
- 신호 후 N일 시장 움직임 vs 벤치마크
- **신뢰구간 + 표본 수 명시** (표본 적어 불확실)

**원칙**: 벤치마크를 못 이기면 "신호 없음"이 정직한 결론. 억지로 의미 부여 금지.

## 5. 신뢰도 표시 (양질의 정보 = 신호 + 신뢰도)

모든 신호에 다음을 **함께** 표시한다:

| 항목 | 내용 |
|---|---|
| 신호 신뢰도 | 해당 회의 confidence (모델이 헷갈렸으면 낮음) |
| 과거 유사사례 | "과거 유사 상황 N건 중 X건 적중" |
| 표본 경고 | 표본 적을 때 "표본 N건, 불확실" 명시 |
| 근거 | 어떤 조건으로 발동됐는지 (투명성) |

> 나쁜 정보: "매수 신호!" / 양질의 정보: "⚠️ 괴리(톤+0.3, 시장−2%), 과거 유사 8건 중 5건서 변동성↑, 단 표본 적음"

## 6. 출력 형태 (사용자가 보는 것)

회의별 리포트 예시(목표):
```
2020-03-15 FOMC
  톤: 긍정 (+0.12, 신뢰도 0.74)   직전 대비 🔼 +0.18
  시장: S&P500 −11.98%, VIX +24.9
  신호: 🔴 괴리 경고 — 연준 톤 긍정 vs 시장 급락
  참고: 과거 위기기 유사 패턴. 표본 적음. 투자조언 아님.
```

## 7. 한계·면책 (반드시 동봉)

- **상관 ≠ 인과** — 톤과 시장의 관계가 인과를 뜻하지 않음
- **표본 작음** (~170회) — 규칙이 과거에 맞아도 미래 보장 못 함
- **톤 ≠ 정책 스탠스** — 감성은 금리 방향이 아니라 경기 톤
- **교란 변수** — 시장은 금리결정·기자회견·지표에도 반응
- **도메인 갭** — 뉴스는 FOMC보다 노이즈 많음
- **연구·참고용. 투자 조언이 아니며, 투자 책임은 본인에게.**
