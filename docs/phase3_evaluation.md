# Phase 3 — 감성 엔진 평가·개선 결과

> 근거: human labeling 150문장(2명 교차) 기반 평가. 잠정(라벨 품질 개선 여지 있음).

## 1. 무엇을 했나

- 작년 파인튜닝 FinBERT를 베이스라인으로 통합 (argmax 라벨이 아니라 **softmax 확률 + 엔트로피** 반환).
- human label 150문장(FOMC 성명문 문장, 4명이 2명씩 교차 라벨링)으로 **독립 평가**.
- 발견된 결함(과신)을 **온도 보정(temperature scaling)**으로 개선.

## 2. 라벨 신뢰도 (정답지 품질)

| 짝 | 일치율 | Cohen's kappa |
|---|---|---|
| 1–75 | 62.7% | 0.43 (낮음, 기준 재합의 권장) |
| 76–150 | 78.7% | 0.65 (양호) |

→ 인간 평가자 간 일치율 자체가 63~79%. FOMC 문장의 본질적 모호성 때문이며, **이 수준이 모델 성능의 현실적 상한(천장)**이다.

## 3. 평가 결과

| 지표 | 베이스라인 | 보정 후 (T=3.1) |
|---|---|---|
| **Macro-F1** | 0.655 | 0.655 (보정은 예측을 안 바꿈) |
| 정확도 | 0.673 | 0.673 |
| **ECE** | **0.257 (과신, 나쁨)** | **0.07~0.10 (양호)** |

- 클래스별 F1: 긍정 0.48 / 중립 0.68 / 부정 0.81
- 주요 오류: 사람이 "중립"이라 한 문장을 모델이 "긍정"으로 과대 예측 (혼동행렬 neu→pos 28건)
- 엔트로피 유효성: 저불확실성 정확도 0.78 vs 고불확실성 0.57 → **엔트로피가 오류 위험을 잘 포착**

## 4. 해석 (정직하게)

- **F1 0.655는 인간 일치율(63~79%) 범위 내** → 모델이 사람만큼 일관됨. FOMC 모호성을 고려하면 "인간 수준 근접".
- 작년 주장(0.84)보다 낮은 건, 우리가 **독립적으로** 재고 정답지에 노이즈(특히 1–75 짝)가 있기 때문.
- **과신(ECE 0.257)은 실제 결함**이었고, 온도 보정으로 0.07~0.10까지 교정. 확신도 가중 인덱스가 정직해짐.

## 5. 개선 내용 (작년 대비)

1. argmax 라벨 → **확률·엔트로피 보존** (작년 결함 수정)
2. **캘리브레이션 적용** (`engine/sentiment.py` TEMPERATURE=3.1) → 과신 교정
3. `model_tag`로 baseline(`finbert-finetuned`) vs 보정(`finbert-cal`) 구분 보관

## 6. 한계·다음

- 정답지 1–75 짝 일치율 낮음 → 기준 재합의 후 재라벨하면 측정이 더 깨끗해짐 (F1 상승 가능).
- F1 추가 개선은 임계값 조정 가능하나, 표본 150개라 과적합 경계.
- 합격 기준(F1·ECE 목표치)은 지도교수와 합의 필요.
- 보정 엔진으로 인덱스 재계산은 Phase 4에서.

## 부록 — 합격 기준 참고치

| 지표 | 일반 기준 | 본 연구 적정(FOMC·인간천장 고려) |
|---|---|---|
| Macro-F1 | 0.7+ 양호 | ~0.70 (인간 일치율 수준이면 충분) |
| ECE | 0.1 미만 양호 | <0.10 (보정으로 달성) |
